# Particle Gesture Engine

用手势操控粒子。打开摄像头，伸手就能把 5000 颗粒子聚拢、炸开、旋转。

## 快速开始

下载 `index.html`，浏览器直接打开，允许摄像头即可。

没有摄像头也能玩 — 自动降级为鼠标交互。

## 六种形态

底部按钮或键盘 `1`-`6` 切换，粒子会平滑过渡到新形态。

| | 形态 | 快捷键 |
|---|---|---|
| ☁ | 星云 | `1` |
| 🧬 | 螺旋 | `2` |
| 🔮 | 星球 | `3` |
| 🌊 | 波浪 | `4` |
| 💍 | 光环 | `5` |
| ✨ | 散射 | `6` |

## 手势操作

| 手势 | 效果 |
|---|---|
| 🤏 捏合（拇指食指靠近） | 粒子聚拢到指尖 |
| 🖐 张开手掌 | 粒子从聚拢点炸开 |
| ✊ 握拳 | 粒子环绕手掌旋转 |

先捏合聚拢、再张开炸开，可以玩出「收→炸」的冲击感。

## 鼠标 / 触摸

- **拖拽** — 旋转视角
- **滚轮** — 缩放
- **按住点击** — 吸引粒子
