# Particle Gesture Engine

伸手就能操控粒子。

打开摄像头，用手势控制 5000 颗粒子聚拢、炸开、旋转。没有摄像头也能玩——自动降级为鼠标交互。

## 玩法

### 启动

```bash
# 任选一种
npx serve .
python3 -m http.server
```

浏览器打开对应地址，允许摄像头即可。必须通过 HTTP 访问，`file://` 下摄像头权限受限。

### 六种形态

底部按钮或键盘 `1`–`6` 切换，粒子平滑过渡。

| 快捷键 | 形态 | 说明 |
|--------|------|------|
| `1` | 星云 | 随机漂浮 |
| `2` | 螺旋 | 旋臂结构 |
| `3` | 星球 | 球面分布 |
| `4` | 波浪 | 正弦起伏 |
| `5` | 光环 | 圆环排列 |
| `6` | 散射 | 粒子炸开 |

还有隐藏形态：输入任意文字（支持中文），粒子会排列成文字形状。

### 手势操作

| 手势 | 效果 |
|------|------|
| 五指张开 | 放大 |
| 握拳 | 缩小 |
| 1 根手指 | 心跳脉冲 |
| 2 根手指 | 正向旋转 |
| 3 根手指 | 反向旋转 |

先放大再缩小，可以玩出呼吸感；配合旋转和心跳，组合出各种效果。

### 鼠标 / 触摸

- 拖拽 — 旋转视角
- 滚轮 — 缩放
- 按住点击 — 吸引粒子

## 技术细节

- 单 HTML 文件，零构建步骤，浏览器直接跑
- Three.js 渲染 + UnrealBloomPass 后处理
- MediaPipe Hands 实时手势识别
- CDN 优先加载，断网自动切换 `libs/` 本地副本
