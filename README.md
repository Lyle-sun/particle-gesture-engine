# Particle Gesture Engine

基于 Three.js + MediaPipe Hands 的实时手势粒子交互系统。通过摄像头捕捉手部动作，驱动 5000 个粒子在 6 种 3D 形态间切换与交互。

## 预览

直接在浏览器打开 `index.html` 即可运行，无需构建。

> 需要摄像头权限用于手势识别；无摄像头时自动降级为鼠标交互。

## 形态

| 形态 | 快捷键 | 说明 |
|------|--------|------|
| 星云 | `1` | 三臂螺旋星云结构 |
| 螺旋 | `2` | 双链 DNA 螺旋 |
| 星球 | `3` | 球面均匀分布 |
| 波浪 | `4` | 网格正弦波面 |
| 光环 | `5` | 环绕核心的环形 |
| 散射 | `6` | 随机空间分布 |

## 手势

| 手势 | 识别方式 | 效果 |
|------|----------|------|
| 捏合 | 拇指与食指靠近 | 粒子被吸引聚拢至指尖 |
| 张开 | 4 指以上展开 | 粒子从聚拢点炸开 |
| 握拳 | 手指收拢 | 粒子环绕手掌旋转 |

手势有 3 帧平滑防抖，避免误触发。吸引→炸开时记录聚拢点作为爆炸中心，形成"先收后炸"的视觉冲击。

## 鼠标 / 触摸

无摄像头时或手势未识别时，可通过鼠标/触摸操作：

- **拖拽**：旋转视角
- **滚轮**：缩放
- **点击拖拽**：吸引粒子（短距拖拽触发）

## 技术细节

- **Three.js 0.160** — WebGL 渲染，ACES 色调映射，加性混合粒子
- **MediaPipe Hands 0.4** — 手部 21 关键点实时追踪，callback 驱动
- **粒子物理** — 弹簧力趋向形态目标 + 手势力场 + 阻尼衰减 + 软边界约束
- **2D→3D 映射** — 手部屏幕坐标通过 Raycaster 反投影到相机朝向平面
- **单文件** — 零依赖安装，CDN 加载，浏览器直接打开
