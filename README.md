# Particle Gesture Engine

用手势操控粒子。打开摄像头，伸手就能把 5000 颗粒子聚拢、炸开、旋转。

## 快速开始

```bash
# 任选一种方式启动本地服务
npx serve .
# 或
python3 -m http.server
```

然后打开浏览器访问 `http://localhost:3000`（serve）或 `http://localhost:8000`（python），允许摄像头即可。

> 需要通过 HTTP 服务访问，`file://` 协议下摄像头权限受限。
> 没有摄像头也能玩 — 自动降级为鼠标交互。

依赖优先从 CDN 加载（jsdelivr），CDN 不可用时自动切换到 `libs/` 下的本地副本，断网也能跑。

## 六种形态

底部按钮或键盘 `1`-`6` 切换，粒子会平滑过渡到新形态。

| | 形态 | 快捷键 |
|---|---|---|
| ☁ | 星云 | `1` |
| 🧬 | 螺旋 | `2` |
| 🔮 | 星球 | `3` |
| 🌊 | 波浪 | `4` |
| 💍 | 光环 | `5` |
| ✨ | 散射 | `6` |

## 手势操作

| 手势 | 效果 |
|---|---|
| 🤏 捏合（拇指食指靠近） | 粒子聚拢到指尖 |
| 🖐 张开手掌 | 粒子从聚拢点炸开 |
| ✊ 握拳 | 粒子环绕手掌旋转 |

先捏合聚拢、再张开炸开，可以玩出「收→炸」的冲击感。

## 鼠标 / 触摸

- **拖拽** — 旋转视角
- **滚轮** — 缩放
- **按住点击** — 吸引粒子
